Build Your Question Bank

Paste your questions in JSON format, load a sample set, or use the AI Prompt Generator to create questions with any AI assistant. Required: question, options, correct_option, explanation.

    How to Write Valid JSON for MedQuiz Pro

    1Start with [ and end with ] — questions must be inside an array.
    2Each question is an object wrapped in { }. Separate multiple objects with commas.
    3Required fields: "question", "options" (array of 4–5 strings), "correct_option", "explanation".
    4Strings use double quotes "like this", not single quotes. Check commas.
    5Use the AI Prompt Generator to have any AI create perfect JSON automatically.
