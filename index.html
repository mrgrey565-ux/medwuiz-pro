Build Your Question Bank

Paste your questions in JSON format, load a sample set, or use the AI Prompt Generator to create questions with any AI assistant. Required fields: question, options, correct_option, explanation.

    How to Write Valid JSON for MedQuiz Pro

    1 Start with [ and end with ] — questions inside an array.
    2 Each question is an object wrapped in { }. Separate multiple objects with commas.
    3 Required: "question", "options" (4 strings), "correct_option" (must match exactly), "explanation".
    4 Always use double quotes "like this". Check for missing or extra commas.
    5 Use the AI Prompt Generator button to auto-generate perfect JSON.
